# SEATLY – Unified Ticket Booking System

SEATLY is a full-stack web application that provides a unified platform for booking movie, train, and flight tickets. The project focuses on delivering a clean, user-friendly, and advertisement-free booking experience through a single interface.

## Features

- Movie Ticket Booking
- Train Ticket Booking
- Flight Ticket Booking
- Dynamic Passenger Management
- Booking History Management
- Cancellation Functionality
- Responsive User Interface
- Backend API Integration
- Database-Driven Booking Storage

## Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Python
- FastAPI

### Database
- MySQL

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/SEATLY-Unified-Ticket-Booking-System.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python main.py
```

4. Open

```text
http://localhost:8000
```
